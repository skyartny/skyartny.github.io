Flex-Check Pro 用户指南

本指南旨在帮助管理员使用 Flex-Check Pro 高效管理学生花名册、追踪出勤情况以及处理数据。

1. 开始使用 (Getting Started)

安装应用图标 (添加到主屏幕)

为了方便快速访问，您可以将 Flex-Check Pro 添加到设备的主屏幕，就像原生应用一样。

应用网址: https://flexcheck.framedghosts.com/

iPhone / iPad (Safari)

  1. 在 Safari 浏览器中打开上述应用链接。
  2. 点击底部或顶部的 分享 图标 (带有向上箭头的方框)。
  3. 向下滚动并点击 添加到主屏幕 (Add to Home Screen)。
  4. 点击 添加 (Add)。

Android (Chrome)

  1. 在 Chrome 浏览器中打开上述应用链接。
  2. 点击右上角的 菜单 图标 (三个点)。
  3. 点击 安装应用 (Install app) 或 添加到主屏幕 (Add to Home Screen)。
  4. 按照提示完成安装。

初始设置

首次启动应用程序时，您必须设置管理员账户以保护花名册数据。

  1. 点击 ROSTER (花名册) 标签页。
  2. 您将看到 Setup Admin (设置管理员) 屏幕。
  3. Create Password: 输入并确认一个安全的密码。
  4. Security Question: 选择一个问题并提供答案。
    • 重要提示: 如果您忘记密码，这是找回账户且不丢失数据的 唯一 方法。
  5. 点击 CREATE ACCOUNT (创建账户)。

2. 学生管理

批量添加学生 (Adding Students)

  1. 在 ROSTER 标签页，点击 + BULK IMPORT (批量导入) 按钮。
  2. 在文本框中输入学生姓名，每行一个。
    • 格式: First Last (例如 John Smith) 或 Last, First (例如 Smith, John)。
  3. 点击 IMPORT LIST (导入列表)。
  4. 处理重复:
    • 如果姓名已存在，系统会暂停并询问：使用带编号的名称导入（例如 "John Smith 2"）或跳过。

编辑与停用

  • 编辑姓名: 在列表中点击学生，然后直接点击顶部的姓名进行编辑。
  • 联系信息: 更新 Parent Email/Phone。退出时自动保存。
  • 停用 (推荐): 切换 Active/Inactive 按钮。从 Check-In 中隐藏学生但保留历史记录。
  • 删除 (永久): 点击 DELETE STUDENT ACCOUNT。 警告: 永久删除所有历史记录。

3. 课时管理

增加课时与调整

  1. 在 ROSTER 中打开学生个人资料。
  2. 找到 Add Classes 部分。
  3. 选择类型（例如 "Refill"）。
  4. 输入课时数（例如 10）。使用负数（例如 -1）来修正错误。
  5. 点击 ADD。

4. 日常操作

签到流程

  1. 进入 CHECK-IN 标签页。
  2. 搜索学生姓名。
  3. 点击黑色的 CHECK IN 按钮。
  4. 系统扣除 1 课时 并标记为已签到。

每日报告

  1. 进入 ROSTER 标签页（需登录）。
  2. 查看顶部的 Daily Report 部分。
  3. 使用日期选择器查看历史记录。
  4. 标题徽章显示当天的总人数。

5. 数据与导出

Excel 导出

下载详细报告用于分析。

  1. 进入 ROSTER > Data Management。
  2. 点击 EXPORT TO EXCEL。
  3. 包含: 学生名单及完整交易历史。

备份与恢复

至关重要: 清除缓存会删除数据！

  1. 定期点击 DOWNLOAD BACKUP。
  2. 安全保存 .json 文件。
  3. 使用 RESTORE BACKUP 恢复数据。

6. 故障排除

忘记密码？

在登录屏幕上点击 Forgot Password? 并回答您的安全问题。

数据丢失 / 应用重置？

如果浏览器缓存被清除，请使用您保存的 .json 文件进行 RESTORE BACKUP。如果没有备份，数据将永久丢失。
